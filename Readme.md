# Event Management System (Frontend)

A frontend-only, data-driven Event Management System built using **HTML, CSS, and JavaScript**.  
This project dynamically displays college symposiums, workshops, and conferences using a separate data file, allowing easy updates **without modifying the main HTML structure**.

The application is deployed as a **static website on Netlify** and does not require any backend or database.

This project solves that problem by **separating event data from the user interface**.

Event details such as:
- Event name
